Quick Dictionary

A Firefox webExtension.

Translate any words with just 1-click.

Support many languages with Wiktionary and Google Translate.

Download for Firefox Source on Github

Welcome to a new Quick Dictionary!

Quick Dictionary v2.0.0 has been redesigned completely. See all new features below.

Features:

  • Quick Button: Selecting any word or paragraph on any website will show up a button. Clicking on it will open a popup, or show the meaning in Sidebar if available.
  • Quick Popup: If Sidebar is not available, a small popup will show up right below your selected text.
  • Sidebar support: Find the Sidebar-toggle button on your browser, then switch the Sidebar mode to Quick Dictionary.
  • Context Menu: Coming soon.
  • Word suggestion: powered by Wiktionary API.
